- **Đường dẫn File:** `README.md`

## Mô tả chi tiết công việc

1. Cập nhật `README.md` với:
   - Mô tả dự án, mục tiêu học thuật.
   - Cài đặt dependencies: `npm install`.
   - Chạy local: `npx hardhat node` + `npm run deploy:local` + `npm run dev`.
   - Docker: `docker compose up --build`.
   - Deploy testnet: hướng dẫn cấu hình `.env` + `npm run deploy:testnet`.
   - Kiến trúc: link tới `docs/`.
2. Đảm bảo toàn bộ scripts trong `package.json` đều có document.

## Đầu ra mong muốn

README hoàn chỉnh, developer mới có thể setup và chạy project trong < 10 phút.
